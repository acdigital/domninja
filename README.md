DOM Ninja
=========

> Frontend developer bookmarklet.


Get it
------

Browse [domninja.com](http://domninja.com) for the bookmarklet code.


Criteria for auditing
---------------------

* Document tags
* Canonical URL
* HTTP requests
* Deprecated tags
* Deprecated attributes
* Iframe tags
* Embed tags
* Scaled img tags
* Conditional comments
* Document comments
* Empty tags
* Empty alt attributes
* Empty href attributes
* Empty src attributes
* Visible tags
* Hidden tags
* Display ratio
* Tags with ID
* Tags with duplicated ID
* Tags with nested ID
* Deep nested tags
* Style tag in body
* External style files
* Inline style attributes
* Third Party styles
* Script tags in head
* External script files
* Inline script attributes
* Script globals
* Third Party scripts
